# 4-bit ALU Verification using SystemVerilog

## Overview
This project implements a 4-bit ALU in SystemVerilog and verifies it using a self-checking verification testbench.

## Supported Operations
- ADD
- SUB
- MUL
- DIV
- NOT
- AND
- OR
- NOR
- NAND
- XOR
- XNOR

## Verification Features
- Randomized stimulus generation
- Constraint-based random verification
- Assertion-based checking
- Expected output model
- Self-checking testbench

## Tools
- Vivado Simulator
- SystemVerilog
